# 🧠 Dopamine Quest - Produtividade Viciante

> "Estude, foque e receba doses de dopamina como num crack digital saudável"

## Sobre o Projeto

Dopamine Quest é um aplicativo web que transforma suas metas diárias em um jogo de recompensas instantâneas. Cada vez que você conclui um bloco de estudo, recebe **caixas surpresa** com pontos, mensagens motivacionais e efeitos visuais de progresso.

## Funcionalidades

- ✏️ Escreva sua meta (ex: "Estudar JavaScript")
- 🧠 Timer focado com contagem regressiva
- 🎁 Caixas surpresa que liberam dopamina visual
- 📊 Barra de progresso e nível do usuário
- 📱 Responsivo: mobile, tablet, desktop
- 🔒 Políticas de Privacidade e Termos de Serviço completos

## Tecnologias

- HTML5, CSS3 (Grid + Flexbox)
- Tailwind CSS (utilitários rápidos)
- Bootstrap 5 (componentes e modais)
- JavaScript Vanilla (sem frameworks)
- LocalStorage para salvar progresso

## Como usar

1. Abra `index.html` no navegador
2. Digite sua meta no campo principal
3. Clique em "Iniciar Jornada"
4. Estude enquanto o timer roda
5. Ao finalizar, abra a caixa surpresa
6. Repita e suba de nível

## PWA - Progressive Web App

O Dopamine Quest é um **PWA completo** e pode ser instalado em:

- ✅ **Android** (Chrome, Edge, Samsung Internet)
- ✅ **iOS/iPadOS** (Safari 16.4+ - via "Adicionar à Tela de Início")
- ✅ **Windows** (Edge, Chrome)
- ✅ **macOS** (Chrome, Edge)
- ✅ **Linux** (Chrome, Edge)

### Funcionalidades PWA:

- 📱 Instalação direta na tela inicial
- 🔄 Funcionamento offline (com cache inteligente)
- 🔔 Push notifications (recompensas e lembretes)
- ⚡ Carregamento ultrarrápido com service worker
- 🎨 Splash screen personalizada
- 📦 Atualizações automáticas

### Como instalar:

**Android/Chrome:**

1. Acesse o site
2. Toque em "Adicionar à tela inicial" ou no botão de instalação que aparece

**iOS/Safari:**

1. Abra no Safari
2. Toque em "Compartilhar" (ícone de quadrado com seta)
3. Selecione "Adicionar à Tela de Início"

**Desktop (Chrome/Edge):**

1. Clique no ícone de instalação na barra de endereço (ou no botão flutuante)
2. Confirme a instalação

### Tecnologias PWA:

- Manifest.json completo
- Service Worker com estratégia Network First
- Cache offline de assets críticos
- Suporte a notificações push
- Background sync para progresso offline
